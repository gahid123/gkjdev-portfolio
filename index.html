Computers; The Beginning and the End
COMPUTERS; THE BEGINNING AND THE END Have you ever imagined how it would be if someone from 1800s somehow teleported to the present? That would be exhilarating! It's fascinating to think how computers evolved over the years. To think that the very first physical computers involved gears or tools bigger than our mobile phones for a single machine to work, I'd say we've come very far. Let us start with Charles Babbage also known as the "father of the computer". He was the key figure in the development of computers. Even though his ideas and designs were not built during his time, it has led to what we are using in the present. Ada Lovelace has also played a major role in shaping the history of computers. With Babbage's designs, Lovelace was able to make the first algorithm which were also crucial in developing today's programming system. Personally and quite evidently, analog computers were the first computers that are closest to what we have today if we talk about the physical image of computers. If you look at an analog computer, you can see the similarity of the setup to today's computers especially personal computers or most commonly known as PCs. An analog computer has a bulky looking screen equivalent to a modern computer's monitor and also a box-looking set of controls that looks like a keyboard. Throughout the years, the progress of computers never stopped. Previous computers were either improved or modified to create more advanced ones for convenience and efficiency. Though improved, it also required a ton amount of systems needed to make computers run properly. In 1943, the Colossus computer was built that looks like it needed a whole garage to fit in. The Colossus computer has hundreds if not thousands of buttons or switches, it may even need a couple of switches to turn it on, unlike modern computers, almost every function just needs a single button to process even having multiple actions for just one command or input. The massive sizes of the electronic machines contained vacuum tubes; in fact, most if not all the first generation computers used vacuum tubes — the glass tube used as electronic switches to perform different functions including the Electronic Numerical Integrator and Computer (ENIAC), the Colossus computer, UNIVAC 1 and others. The end of vacuum tube computers was not until the late 1950s and more advanced tools were used to create faster computers. We've come so far, we've gone through dramatic changes from having computers as big as a room or even a house to computers being held and even smaller than our hands. All these progress, all these evolutions will not happen if it were not for the brilliant minds that kept on working all throughout these upgrades. The computers we have now are a product of a long history of evolution which was made possible by the human mind. We may have come so far, we have not even reached the peak of technology but no matter how advanced our technology is, nothing will ever replace the human brain's brilliance. Everything started from the mind and eventually, everything will end with it.

package com.mycompany.decimaltobinary;

/**
 *
 * @author gasto
 */
import java.util.Scanner;//The Scanner class is imported to allow the program
                        //to read input from the user through the keyboard.

public class DecimalToBinary {//The class is named DecimalToBinary, file name
    public static void main(String[] args) {//The main method is the entry
                                            //point of the program where execution begins.
        
        Scanner scanner = new Scanner(System.in);//A Scanner object is created to read user input.
        String input;                 //The variable input stores what the user types as a string.
        
        //LOOP STATEMENT
        while (true) {    //The program runs inside an infinite loop so the user can convert multiple numbers.
            System.out.print("Enter a decimal number \n(or type STOP to end): ");
            input = scanner.nextLine(); //string input
            if (input.equalsIgnoreCase("STOP")) {
                System.out.println("Program terminated.");
                break;//When the user types STOP, the break statement exits the infinite loop.
            }
            int number = Integer.parseInt(input); //The input string is converted into an integer 
            int decimal = number;                //so mathematical operations can be performed.
            String binary = "";

            //CONVERSION
            if (decimal == 0) {     //If the number is 0, the binary result is simply "0".
                binary = "0";       //Otherwise, the program repeatedly divides the number by 2.
            } else {                //The remainder (0 or 1) from each division is added to the front of the binary string.
                while (decimal > 0) {
                    int remainder = decimal % 2;
                    binary = remainder + binary;
                    decimal = decimal / 2;
                }
            }
            System.out.println("Binary equivalent: " + binary);//prints the output from decimal to binary number
        }
        scanner.close();//closes the scanner to free system resources
    }
}
Quest 2.2 — Radix Converter
RadixConverter_Gahid.java
// Import Scanner class to allow user input from the keyboard
import java.util.Scanner;

public class RadixConverter_Gahid {

    public static void main(String[] args) {
        // Scanner used to read user input from the keyboard
        Scanner input = new Scanner(System.in);

        // Loop allows the program to run repeatedly until the user types STOP
        while (true) {
            // ===== INPUT: Get origin base from the user =====
            System.out.print("Enter origin base (2-16) or STOP to end: ");
            String baseInput = input.nextLine();

            // ===== CONTROL: Check if user wants to terminate the program =====
            if (baseInput.equalsIgnoreCase("STOP")) {
                // Handles non-numeric base input
                System.out.println("Program terminated.");
                break;
            }

            int originBase;

            // ===== VALIDATION: Ensure base input is a valid integer =====
            try {
                originBase = Integer.parseInt(baseInput);
            } catch (Exception e) {
                System.out.println("Invalid base input.");
                continue;
            }

            // ===== VALIDATION: Check that base is within allowed range (2–16) =====
            if (originBase < 2 || originBase > 16) {
                System.out.println("Base must be between 2 and 16.");
                continue;
            }

            // ===== INPUT: Get number to be converted =====
            System.out.print("Enter number in base " + originBase + ": ");
            String numberInput = input.nextLine();

            int decimalValue;

            // ===== CONVERSION: Convert input number from origin base to decimal =====
            try {
                // parseInt automatically validates digits for the specified base
                decimalValue = Integer.parseInt(numberInput, originBase);
            } catch (Exception e) {
                // Handles invalid digits for the given base
                System.out.println("Invalid number for the given base.");
                continue;
            }

            // ===== OUTPUT: Display conversion header =====
            System.out.println("\nConversions of " + numberInput + " (Base " + originBase + "):");

            // ===== OUTPUT: Convert decimal value to bases 2 through 16 =====
            for (int base = 2; base <= 16; base++) {
                // Convert decimal value to the target base and format output
                String convertedValue = Integer.toString(decimalValue, base).toUpperCase();
                System.out.println("Base " + base + ": " + convertedValue);
            }
            // Separator for readability between conversions
            System.out.println("---------------------------------------------");
        }
        // Close scanner resource
        input.close();
    }
}
Quest 3.1 — Bitwise Calculator
BitwiseCalculator_G8.java
// Import Scanner to allow user input from the keyboard
import java.util.Scanner;

public class BitwiseCalculator_G8 {

    public static void main(String[] args) {
        // Create a Scanner object to read user input from the keyboard
        Scanner scanner = new Scanner(System.in);
        
        // Declare variables for the two operands and the result
        int num1, num2, result;
        String operator;

        // INPUT SECTION
        // Ask the user for the first decimal number
        System.out.print("Enter first decimal number: ");
        num1 = scanner.nextInt();
        // Ask the user for the second decimal number
        System.out.print("Enter second decimal number: ");
        num2 = scanner.nextInt();

        // Ask the user for the bitwise operator
        System.out.print("Enter operator (&, |, ^, <<, >>): ");
        operator = scanner.next();

        // PROCESSING / LOGIC SECTION
        // Use a switch statement to identify which operation to perform
        switch (operator) {
            // Bitwise AND operation
            case "&":
                result = num1 & num2;
                System.out.println("Result: " + num1 + " & " + num2 + " = " + result);
                break;
            // Bitwise OR operation
            case "|":
                result = num1 | num2;
                System.out.println("Result: " + num1 + " | " + num2 + " = " + result);
                break;
            // Bitwise XOR operation
            case "^":
                result = num1 ^ num2;
                System.out.println("Result: " + num1 + " ^ " + num2 + " = " + result);
                break;
            // Bitwise LEFT SHIFT operation
            case "<<":
                result = num1 << num2;
                System.out.println("Result: " + num1 + " << " + num2 + " = " + result);
                break;
            // Bitwise RIGHT SHIFT operation
            case ">>":
                result = num1 >> num2;
                System.out.println("Result: " + num1 + " >> " + num2 + " = " + result);
                break;
                
            // ERROR HANDLING SECTION
            // If the operator is not one of the valid bitwise operators
            default:
                System.out.println("Unidentified Operation");
        }
        // Close the scanner to prevent resource leaks
        scanner.close();
    }
}
Quest 3.2 — Arithmetic Calculator
ArithmeticCalculator.java
package com.mycompany.arithmeticcalculator;

/**
 *
 * @author gasto
 */
import java.util.Scanner;

public class ArithmeticCalculator {

    public static void main(String[] args) {

        Scanner input = new Scanner(System.in);

        // Accept user inputs
        System.out.print("Enter first operand: ");
        String operand1 = input.nextLine();

        System.out.print("Enter second operand: ");
        String operand2 = input.nextLine();

        System.out.print("Enter operator (+, -, *, /): ");
        String operator = input.nextLine();

        System.out.print("Enter base (2-16): ");
        int base = input.nextInt();

        // Validate base
        if (base < 2 || base > 16) {
            System.out.println("Invalid base. Base must be between 2 and 16.");
            return;
        }

        try {
            // Convert operands from given base to decimal
            int num1 = Integer.parseInt(operand1, base);
            int num2 = Integer.parseInt(operand2, base);

            int result = 0;
            boolean validOperation = true;

            // Identify operator and perform operation
            switch (operator) {
                case "+":
                    result = num1 + num2;
                    break;
                case "-":
                    result = num1 - num2;
                    break;
                case "*":
                    result = num1 * num2;
                    break;
                case "/":
                    if (num2 == 0) {
                        System.out.println("Error: Division by zero");
                        return;
                    }
                    result = num1 / num2; // integer division
                    break;
                default:
                    System.out.println("Unidentified Operation");
                    validOperation = false;
            }

            if (validOperation) {
                // Convert result back to original base
                String finalResult = Integer.toString(result, base).toUpperCase();
                System.out.println("Result: " + finalResult);
            }

        } catch (NumberFormatException e) {
            System.out.println("Invalid number for the given base.");
        }

        input.close();
    }
}
